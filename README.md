# Nmap Scanning Lab

This project contains basic Nmap scans performed for learning and documentation purposes.  
It demonstrates different types of scans, how they work, and what kind of information can be gathered from a target system.

---

## Objectives
- Understand essential Nmap commands
- Perform different types of network scans
- Interpret scan results
- Document findings and key observations

---

## Scans Included
Located in the **/scans** folder:

- `basic-scan.txt` — Standard TCP scan  
- `service-version-scan.txt` — Detects service versions running on open ports  
- `os-detection.txt` — Attempts OS fingerprinting  
- `common-ports-scan.txt` — Scans frequently used ports  
- `vulnerability-scan.txt` — Runs basic NSE vulnerability scripts
